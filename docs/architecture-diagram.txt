  ── PRIVACY GUARANTEE ───────────────────────────────────────────────────────────────────
    The verifier learns ONLY that the predicate holds.  DOB · country · name · document
    signature · salt  are all private witness inputs and never appear in public inputs.
    An invalid predicate (underage · wrong country · forged sig) makes the circuit
    UNSATISFIABLE  →  no valid Groth16 proof can ever exist.

  ── LAYERS ──────────────────────────────────────────────────────────────────────────────
    [ off-chain ]   Issuer signing  ·  Holder prover (circom + snarkjs)
    [ on-chain  ]   Soroban verifier:  BN254 pairing + issuer/root binding + nullifier + gate
    [ circuits  ]   AgeGte · JurisdictionAllowed · ValidOwner · CredentialAge (flagship)
